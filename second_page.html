HTML CSS JavaScript

HTML

  • язык гипертекстовой разметки
  • позволяет создать каркас веб-страницы
  • элементы html - теги в угловых скобках
  • теги не отображаются на странице, в отличие от их содержимого
  • обязательный элемент веб-приложения

CSS (Cascading Style Sheets)

  • формальный язык описания внешнего вида веб-страницы
  • стилизирует html теги
  • элементы css: селекторы, свойства и их значения
  • имеет принципы наследования и каскадирования

JavaScript

  • динамически типизироанный язык программирования
  • наиболее широкое применение находит в браузерах для придания интерактивности веб-страницам
  • синтаксические форма наименования: CamelCase
  • один из самых популярных языков программирования
